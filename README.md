# ARM Assembly Journey

Learning ARM assembly language from scratch — this repo documents my progress,
practice exercises, and small projects as I build up my skills.

## Structure

- `01-basics/` — early practice programs (hello world, simple math, etc.)
- `02-projects/` — more complete, polished programs

## Tools used

- WSL (Ubuntu) for writing and testing code
- GNU ARM toolchain (`as`, `ld`) for assembling and linking
- QEMU for emulating ARM binaries on x86

## Progress

- [x] Hello world / basic exit codes
- [x] Simple math operations
- [ ] Loops and branching
- [ ] Functions and stack usage
- [ ] Working with arrays
